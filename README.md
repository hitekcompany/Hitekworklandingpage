# Hitek Work - Landing Page

This is a code bundle for Hitek Work - Landing Page. The original project is available at https://www.figma.com/design/3OQZxRvo9LGJ9464FYeVoS/Hitek-Work---Landing-Page.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deployment

Sau này nếu team mình có thay đổi gì trên source code, chỉ cần push commit lên nhánh `main` của repo GitHub, Vercel sẽ tự động deploy phiên bản mới.
