# Hitek Work - Landing Page

This is a code bundle for Hitek Work - Landing Page. The original project is available at https://www.figma.com/design/3OQZxRvo9LGJ9464FYeVoS/Hitek-Work---Landing-Page.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deployment

In the future, if our team makes any changes to the source code, simply push the commit to the `main` branch of the GitHub repository. Vercel will automatically deploy the new version.
